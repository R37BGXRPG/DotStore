# DotStore 🍿🚀✨ - v1.0 "Wheelchair"

DotStore is a premium, dedicated Android App Store for **JusDots** applications (DotNotes, JusBrowse, JusChatz). It ensures users always have access to the latest releases directly from GitHub.

![Version](https://img.shields.io/badge/Version-1.0%20%22Wheelchair%22-violet?style=for-the-badge&logo=android)
![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)

## Features 💎

- **Aggressive Detection**: Automatically finds JusDots apps regardless of package name rebranding (supports `shubh`, `jusdots`, and `jus` variants).
- **One-Tap Downloads**: Integrated with GitHub Releases API for seamless APK fetching.
- **Robust Installer**: Modern URI handling ensures system installation dialogs appear instantly.
- **Premium UI**: Sleek dark-themed design using Jetpack Compose with stylized typography.
- **Stability First**: Crash-proof font loading and optimized background tasks.

## Apps Supported 🆘

1. **DotNotes**: Elegant note-taking.
2. **JusBrowse**: Privacy-focused browsing.
3. **JusChatz**: Real-time messaging.

## Getting Started 🍿🚀

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/R37BGXRPG/DotStore.git
   ```
2. **Setup**: Open the project in Android Studio (Ladybug or later).
3. **Build**: Build and install the APK on an Android device (API 30+ recommended).

## Technical Stack 🛠️

- **Language**: Kotlin
- **UI**: Jetpack Compose (Material 3)
- **Networking**: Retrofit & OkHttp
- **Image Loading**: Coil
- **Concurrency**: Kotlin Coroutines & Flow
- **Background Work**: WorkManager
